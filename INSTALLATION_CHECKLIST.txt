# Installation Checklist for Employee Management System

## Pre-Installation Requirements

### 1. Software to Install
- [ ] Node.js (v14 or higher) - https://nodejs.org/
- [ ] Git (optional, for version control) - https://git-scm.com/

### 2. Accounts to Create
- [ ] MongoDB Atlas account - https://cloud.mongodb.com/
  - [ ] Create free cluster
  - [ ] Create database user (username: root, password: root)
  - [ ] Whitelist IP address (0.0.0.0/0 for development)
  - [ ] Get connection string

---

## Installation Steps

### Step 1: Verify Node.js Installation
```cmd
node --version
npm --version
```
Expected: Node v14+ and npm v6+

### Step 2: Navigate to Project Directory
```cmd
cd C:\EcomClone
```

### Step 3: Install All Dependencies
```cmd
npm run install-all
```

**This will install:**
- Backend dependencies (8 packages + 2 dev packages)
- Frontend dependencies (React and related packages)

**Time estimate:** 2-5 minutes depending on internet speed

## Post-Installation Setup

### Step 4: Configure Environment Variables
Your `.env` file should already have:
```env
MONGO_URI=mongodb+srv://root:root@emp-management.ewvvtkn.mongodb.net/employee_management?retryWrites=true&w=majority
JWT_SECRET=your_super_secret_jwt_key_here
PORT=5000
NODE_ENV=development
CLIENT_URL=http://localhost:3000
```

### Step 5: Start the Application
```cmd
npm run dev
```

You should see:
```
[0] Server running in development mode on port 5000
[0] MongoDB Atlas Connected: emp-management-shard-00-00...
[1] webpack compiled successfully
[1] Compiled successfully!
```

### Step 6: Access the Application
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000

### Step 7: Create First User
1. Go to http://localhost:3000
2. Click "Register here"
3. Fill in details and select role: "Admin"
4. Click Register

## If Installation Fails

### Clear everything and reinstall:
```cmd
# Delete node_modules folders
rmdir /s /q node_modules
rmdir /s /q client\node_modules

# Delete package locks
del package-lock.json
del client\package-lock.json

# Clear npm cache
npm cache clean --force

# Reinstall
npm run install-all
```

### Check for common issues:
1. **Node.js not installed** â†’ Install from https://nodejs.org/
2. **Old Node.js version** â†’ Update to v14 or higher
3. **Port already in use** â†’ Kill process or change port in .env
4. **MongoDB connection failed** â†’ Check IP whitelist in Atlas
5. **Permission denied** â†’ Run Command Prompt as Administrator
